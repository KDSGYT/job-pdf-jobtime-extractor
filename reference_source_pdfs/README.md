# Reference source files

Store the seven reusable source workbooks/reports for DEXTR here.

Use these files as the standing source set while developing and testing the web app. The eighth target workbook is intentionally not stored here because it may change each run.

Expected use:

1. Keep the seven source files in this folder.
2. Keep their filenames stable so the app/tests can refer to them consistently.
3. Upload or provide the eighth target workbook separately each time.

Do not put generated output workbooks or temporary exports in this folder.
